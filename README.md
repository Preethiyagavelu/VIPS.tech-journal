# Research Portal Clone

A pixel-faithful layout and interaction clone of IEEE Xplore homepage with a custom color palette.

## Features
- Responsive layout using Bootstrap 5
- GSAP animations for interactive elements
- Semantic HTML with accessibility features
- Custom color palette implementation
- Placeholder content and images

## Tech Stack
- HTML5
- Bootstrap 5 (CSS & JavaScript)
- Vanilla JavaScript
- GSAP (GreenSock Animation Platform)
- Bootstrap Icons

## Installation & Usage
1. Clone this repository or download the files
2. Open `index.html` in any modern web browser
3. No server required - all functionality works locally

## Customization
- Colors: Modify variables in `assets/css/bootstrap-overrides.css`
- Content: Update `assets/js/main.js` researchData array
- Styles: Edit `assets/css/styles.css`
